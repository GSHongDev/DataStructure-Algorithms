# 数据结构与算法仓库

## 项目结构
```
DataStructure/
├── Heap/          # 堆实现
│   ├── Heap.c     # 源文件
│   └── Test.c    # 测试用例
├── Queue/        # 队列实现
├── Stack/        # 栈实现
├── SList/        # 单向链表
└── SeqList/      # 顺序表

## 构建说明
使用Visual Studio编译运行测试文件

## 模块功能
每个数据结构包含：
- 头文件(.h)
- 实现文件(.c)
- 测试文件(Test.c)
